# Rails 4GL

This is add-on gem to rails to implement 4th generational language programming

## Installation

Add this line to your application's Gemfile:

    gem 'rails_4gl'

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install rails_4gl
